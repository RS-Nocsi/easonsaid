EasonSaid 开放接口

由一百多位EF分享的随机Eason歌词、高清壁纸、大头仔图标 —— 简单的 HTTP 请求即可接入你的项目，无需认证，永久免费，完全开源。

可用接口

所有接口均支持跨域调用（CORS），无需 API Key

🎵

随机歌词

随机返回一条Eason歌词分享，包含分享者、歌词内容和出处信息。支持纯文本和 JSON 两种格式。

GET /output.php
🖼️

横屏壁纸

随机返回一张横屏（landscape）高清壁纸图片，不缓存，直接以图片流输出，可用于网页背景等场景。

GET /landscape_images.php
📱

竖屏壁纸

随机返回一张竖屏（portrait）高清壁纸图片，适用于手机壁纸或移动端背景。

GET /portrait_images.php
😄

大头仔图标

随机返回某个城市的对应大头仔图标，可用于网站图标等（就像本站一样）

GET /datouzai_icon.php
📖
所有接口均支持跨域（CORS），永久开源免费。接口详细用法请查看 API 文档。
